# KAN-VAE
KAN-VAE: Integrating Kolmogorov-Arnold Networks with Variational Auto-Encoders


##
 ðŸ“‹ Overview
 This repository contains the implementation of **KAN-VAE** a novel architecture that combines Kolmogorov-Arnold Networks with Variational Auto-Encoders(VAEs). By replacing traditional Multi-Layer Perceptrons (MLPs) in VAE encoders and decoders with KAN layers, we investigate the potential for improved generative capabilities and parameter efficiency.
